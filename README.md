# T3 Monorepo (prisma)

Monorepo built from t3 stack including:

- Tailwind
- tRPC
- Prisma
- NextJS App router
- PostgreSQL

Inspired by and based on https://www.jumr.dev/blog/t3-turbo and https://github.com/t3-oss/create-t3-turbo

## Getting started

### Dependencies

- pnpm (version = ^9.7.1)
- node (version = ^20.16.0)
- jq
- docker

### Start developing

```
## Initialise the repository
./init_repo.sh

## Start developing :)
pnpm turbo dev
```
